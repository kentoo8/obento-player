# Multi Video Player

複数の動画ファイルをグリッド表示し、指定した時間ごとにランダムなシーンへ自動切り替えするブラウザベースのプレイヤー。

## 起動方法

`index.html` をダブルクリックするだけ。ローカル環境（`file://`）で完全に動作し、サーバーは不要です。

## 使い方

1. 画面内に動画ファイルをドラッグ＆ドロップ（または「追加」ボタンから選択）
2. **自動的に音声付きで再生が開始され**、指定秒数ごとにランダムなシーンへ切り替わります。

> **ヒント**: モダンブラウザ（Chrome/Edgeなど）であれば、動画フォルダをそのままドラッグ＆ドロップして中身を一括読み込みすることが可能です。

## コントロールバー

マウスを画面上端に移動するとコントロールバーが表示されます。動画が読み込まれていない初期状態では常時表示されます。

| 操作 | 説明 | ショートカット |
|---|---|---|
| **App ロゴ** | クリックで全リセット（初期状態に戻す） | — |
| **切替間隔** | シーンが切り替わるまでの秒数（5〜300秒） | — |
| **レイアウト** | 1 / 2 / 4 / 6 / 8 / 9 画面を選択 | — |
| **再生速度** | 0.5x 〜 3.0x までの再生速度を選択 | — |
| **再生 / 停止** | 全セルの再生を一括制御 | `Space` |
| **音声ON/OFF** | 全セルの音声をまとめて ON/OFF | — |
| **音量スライダー** | 全体の音量調整（デフォルト80%） | `↑` / `↓` |
| **追加** | ファイルを追加読み込み | — |
| **シャッフル** | 手動で全セルを即座にランダムシーン切替 | `S` |
| **回転切替** | 縦動画を自動的に270度回転して横長に表示する機能のON/OFF | — |

## セル内個別コントロール

動画セル内の下部に表示されるコントロールバーから以下の操作が可能です。

| ボタン | 説明 | ショートカット |
|---|---|---|
| **ミュート** | そのセル単体の音声をミュート / 解除 | — |
| **⏮ / ⏭** | そのセルで再生する動画を前後に切り替え | — |
| **10秒戻る / 進む** | 再生位置を10秒前後にシーク | — |
| **▶ / ⏸** | そのセル単体の再生を一時停止 / 再開 | — |
| **フォーカス** | そのセルを全画面表示（単体集中モード） | `F` (入/切) / `Esc` (切) |
| **ピン留め** | 自動切替を停止し、同じ動画をループ再生 | — |
| **除外** | `Alt` を押しながらホバーで表示。その動画を以降の再生候補から除外する | — |



## 対応ファイル形式

ブラウザが再生できる形式のみ対応（非対応ファイルは自動的にスキップ）。

| 形式 | 対応 |
|---|---|
| `.mp4` (H.264) | ✅ 推奨・全ブラウザ |
| `.webm` (VP8/VP9) | ✅ Chrome / Firefox / Edge |
| `.ogg` / `.ogv` | ✅ Firefox 中心 |
| `.mov` / `.avi` / `.mkv` | ❌ ブラウザに依存（再生できない場合があります） |
