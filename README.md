# Multi Video Player

複数の動画ファイルをグリッド表示し、指定した時間ごとにランダムなシーンへ自動切り替えするブラウザベースのプレイヤー。

## 起動方法

`index.html` をダブルクリックするだけ。サーバー不要。

## 使い方

1. 動画ファイルをウィンドウにドラッグ＆ドロップ（または「追加」ボタン）
2. 自動的に再生開始。指定秒数ごとにランダムなシーンへ切り替わる
3. 音声を出したい場合は「ミュート」ボタンをクリック → 「音声ON」になる

## コントロール

| 操作 | 説明 |
|---|---|
| **切替間隔** | シーンが切り替わるまでの秒数（1〜300秒） |
| **レイアウト** | 1 / 2 / 4 / 6 / 9 画面を選択 |
| **ミュート / 音声ON** | 全セルの音声をまとめて ON/OFF |
| **音量スライダー** | 音量調整（操作すると自動で音声ON） |
| **再生 / 停止** | 全セルの再生を一括制御 |
| **シャッフル** | 手動で全セルを即座にランダムシーン切替 |
| セル右上の 🔇 | そのセル単体のミュート切替 |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | 再生 / 停止 |
| `S` | シャッフル |
| `↑` / `↓` | 音量 +5% / -5% |

## 対応ファイル形式

ブラウザが再生できる形式のみ対応（非対応ファイルは自動的にスキップ）。

| 形式 | 対応 |
|---|---|
| `.mp4` (H.264) | ✅ 推奨・全ブラウザ |
| `.webm` (VP8/VP9) | ✅ Chrome / Firefox / Edge |
| `.ogg` / `.ogv` | ✅ Firefox 中心 |
| `.mov` / `.avi` / `.mkv` | ❌ ブラウザ非対応 |

> **注意**: ブラウザの自動再生ポリシーにより、起動直後は全動画がミュート状態になります。「ミュート」ボタンをクリックするか、音量スライダーを操作すると音声が有効になります。
