# Obento Player

<img src="assets/preview.png" width="800" alt="Obento Player Preview">

複数の動画をお弁当箱の仕切りのようにグリッド配置し、指定した時間間隔ごとにランダムなシーンへ自動切り替えするブラウザベースのプレイヤーです。

## 特徴

- 指定した時間間隔（5秒〜300秒）で、各動画のランダムな位置を自動抽出。
- 自由に動画レイアウトの切り替え可能（1枚〜9枚）。
- UIを隠す没入モードや、特定動画の全画面表示に対応。
- 速度、ミュート、シャッフル等のキーボードショートカットに対応。

## ダウンロード方法

1. 画面右上の緑色のボタン **[<> Code]** をクリックします。
2. **[Download ZIP]** を選択してファイルをダウンロードします。
3. ダウンロードされた ZIP ファイルを解凍（展開）します。

## 起動方法

解凍したフォルダの中にある `index.html` をダブルクリックするだけです。ローカル環境（`file://`）で完全に動作し、サーバーは不要です。

## 使い方

1. 画面内に動画ファイルをドラッグ＆ドロップ（または「追加」ボタンから選択）
2. **自動的に音声付きで再生が開始され**、指定秒数ごとにランダムなシーンへ切り替わります。

> **ヒント**: モダンブラウザ（Chrome/Edgeなど）であれば、動画フォルダをそのままドラッグ＆ドロップして中身を一括読み込みすることが可能です。

## コントロール（サイドバー）

画面左側のサイドバーからアプリ全体の操作を行います。`F`キーでシアターモードに切り替えると、サイドバーを隠して動画のみに集中できます。

| 操作 | 説明 | ショートカット |
|---|---|---|
| **App ロゴ** | クリックで全リセット（初期状態に戻す） | — |
| **再生 / 停止** | 全セルの再生を一括制御 | `Space` |
| **再生速度** | 0.5x 〜 3.0x までの再生速度を選択 | `[` / `]` |
| **音声 ON/OFF** | 全体の音量を調整（ホバーでスライダー表示） | `M` / `↑` `↓` |
| **レイアウト** | 1 / 2 / 4 / 6 / 8 / 9 画面を選択 | — |
| **シャッフル** | 手動で全セルを即座にランダムシーン切替 | `S` |
| **切替間隔** | 詳細メニュー内。シーンが切り替わるまでの秒数（5〜300秒） | — |
| **追加** | 詳細メニュー内。ファイルを追加読み込み | — |
| **シアターモード** | 詳細メニュー内。サイドバーを隠して動画のみを表示 | `F` |
| **回転切替** | 詳細メニュー内。縦動画を自動回転して横長に表示。通常クリックで270度回転、Alt+クリックで90度回転。 | — |

## セル内個別コントロール

動画セル内の下部に表示されるコントロールバーから以下の操作が可能です。

| ボタン | 説明 | ショートカット |
|---|---|---|
| **⏮ / ⏭** | そのセルで再生する動画を前後に切り替え | — |
| **10秒戻る / 進む** | 再生位置を10秒前後にシーク | `←` / `→` (ホバー中) |
| **▶ / ⏸** | そのセル単体の再生を一時停止 / 再開 | — |
| **フォーカス** | そのセルを全画面表示 | `F` (入/切) / `Esc` (切) |
| **ピン留め** | `Alt` を押しながらホバーで表示。自動切替を停止 | `P` (ホバー中) |
| **ソロ再生** | その動画の音だけを出す。再度クリックで解除 | — |
| **除外** | `Alt` を押しながらホバーで表示。その動画を候補から削除 | — |

## 対応ファイル形式

| 形式 | 対応 |
|---|---|
| `.mp4` (H.264) | ✅ 推奨・全ブラウザ |
| `.webm` (VP8/VP9) | ✅ Chrome / Firefox / Edge |
| `.ogg` / `.ogv` | ✅ Firefox 中心 |
| `.mov` / `.avi` / `.mkv` | ❌ ブラウザに依存 |

非対応動画は ffmpeg などで .mp4 に変換することをオススメします。
